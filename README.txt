국기 색깔 맞추기 게임 PWA 패키지

구성 파일
- index.html: 게임 본문
- manifest.json: 앱 이름, 아이콘, 실행 방식 설정
- service-worker.js: 오프라인 실행을 위한 캐시 설정
- icons/icon-192.png, icons/icon-512.png: 앱 아이콘

사용 방법
1. 이 폴더 전체를 GitHub Pages, Netlify, Vercel 등 HTTPS가 지원되는 웹호스팅에 업로드합니다.
2. 휴대폰 Chrome 또는 Edge에서 접속합니다.
3. 브라우저 메뉴에서 '앱 설치' 또는 '홈 화면에 추가'를 선택합니다.
4. 설치 후 홈 화면 아이콘으로 실행합니다.
