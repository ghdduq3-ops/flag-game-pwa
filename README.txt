# 소희의 50가지 세상 퍼즐 - 그림 표시 수정본

## 수정 내용
- SVG 이미지 생성 인코딩 방식을 안정화했습니다.
- PWA 서비스워커 캐시 버전을 변경하여 기존 캐시가 남아 그림이 안 보이는 문제를 줄였습니다.
- 그림 생성 오류가 있을 경우 화면 하단 안내 문구에 오류를 표시하도록 했습니다.

## 업로드 방법
1. ZIP 압축을 풉니다.
2. 모든 파일을 같은 폴더에 업로드합니다.
3. HTTPS 환경에서 index.html을 실행합니다.
4. 기존 버전이 설치되어 있었다면 브라우저 캐시 삭제 또는 앱 삭제 후 재설치하세요.

## 중요
기존 PWA를 이미 설치한 기기에서는 이전 Service Worker 캐시 때문에 예전 파일이 계속 보일 수 있습니다.
그 경우 브라우저 캐시를 삭제하거나 기존 앱을 삭제 후 다시 설치하세요.
