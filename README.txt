# 소희의 100가지 세상 퍼즐 - 전혀 다른 그림판 PWA

## 수정 내용
- 첨부 HTML을 기준으로 수정했습니다.
- 동물 캐릭터 15종을 각각 다른 실루엣으로 변경했습니다.
- 사람 캐릭터 10종을 의상, 소품, 몸 형태가 다르게 보이도록 변경했습니다.
- 차량/탈것 15종을 차체 구조, 바퀴, 창문, 장비가 다르게 보이도록 변경했습니다.
- PWA 업로드용 파일을 함께 구성했습니다.

## 업로드 방법
1. ZIP 압축을 풉니다.
2. 모든 파일을 같은 폴더에 업로드합니다.
3. HTTPS 환경에서 index.html을 실행합니다.
4. 기존 PWA를 설치한 적이 있으면 앱 삭제 또는 브라우저 캐시 삭제 후 다시 접속하세요.

추가 수정: 각 그림 하단의 작은 그림(이모지 라벨)을 제거했습니다.

추가 수정: 전체 일러스트를 더 고급스럽게 다듬고, 캐릭터 얼굴·의상·장식 디테일을 예쁘게 개선했습니다.

추가 수정: 캐릭터에 팔과 다리를 추가하고, 여아 취향의 더 화사한 파스텔 색감으로 조정했습니다.
